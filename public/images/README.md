# Images Folder

Place your architecture diagrams and blog images here.

## Usage in Blog Posts

Reference images in your markdown files:

```markdown
![Architecture Diagram](/images/your-diagram.png)
```

## Supported Formats

- PNG
- JPG/JPEG
- SVG
- GIF

## Best Practices

- Use descriptive filenames: `aws-kendra-architecture.png`
- Optimize images before uploading (compress for web)
- Recommended max width: 1200px
- Use PNG for diagrams, JPG for photos
